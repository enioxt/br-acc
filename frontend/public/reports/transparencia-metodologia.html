1. Metodologia

A EGOS Inteligencia opera com uma metodologia simples e transparente em 5 etapas:

Etapa 1: Coleta — Apenas Dados Publicos

Todos os dados sao provenientes de fontes publicas oficiais: Portal da Transparencia, Receita Federal (CNPJ publico), TSE, CNJ, CGU, ICIJ (Panama Papers), OpenSanctions, entre 79+ fontes. Nenhum dado privado, hackeado ou obtido ilegalmente.

Etapa 2: Transformacao — Pipelines ETL

46 pipelines automatizados (Python) baixam, limpam e normalizam os dados. CPFs de nao-PEP sao mascarados conforme LGPD. Dados sao convertidos para o formato de grafo (nos + relacionamentos).

Etapa 3: Armazenamento — Grafo de Conhecimento (Neo4j)

Entidades (pessoas, empresas, contratos, sancoes) conectam-se por relacionamentos (SOCIO_DE, DOOU, VENCEU, SANCIONADA). Isso permite cruzamentos impossiveis em tabelas — como descobrir que o socio de uma empresa sancionada doou para o deputado autor da emenda que beneficiou aquela empresa.

Etapa 4: Analise — IA + 26 Ferramentas

Um agente de IA consulta 26 ferramentas em tempo real: busca no grafo Neo4j, APIs publicas (Portal Transparencia, DataJud, Querido Diario, PNCP, Camara, Senado), e busca web. O agente cruza informacoes e gera respostas com cadeia de fontes rastreavel.

Etapa 5: Publicacao — Aberta e Verificavel

Todo codigo e open source (AGPL-3.0). Relatorios incluem fontes consultadas, custo da analise e limitacoes. Nenhuma acusacao — apenas dados publicos cruzados.

2. Principios Eticos

Declaracao de Neutralidade: A EGOS Inteligencia nao faz acusacoes. Apresentamos dados publicos cruzados para que cidadaos, jornalistas e orgaos de controle tirem suas conclusoes. Padroes detectados sao indicios, nao provas.
  • Dados Publicos Apenas: Nenhuma fonte privada, vazada ou ilegal
  • LGPD Compliance: CPFs mascarados (***.***.***-XX). Registro de Tratamento Art. 37 publicado
  • Neutralidade Politica: Algoritmos aplicados uniformemente. CI audita linguagem tendenciosa
  • Open Source (AGPL-3.0): Codigo em github.com/enioxt/EGOS-Inteligencia
  • Rastreabilidade: Cada resposta inclui fontes consultadas e custo
  • Autofinanciado: ~$36/mes (VPS $35 + dominio $1), sem patrocinadores, sem grants

7. Roadmap

Fase 1 — Fundacao ✔ CONCLUIDA

36 fontes carregadas, 9.2M nos, 34.5K relacionamentos. Frontend publico, 2 bots (Discord + Telegram), API com 26 tools, 10 detectores de padrao. 46 pipelines ETL.

Fase 2 — Expansao Politica (em andamento)

Integrar dados eleitorais completos (TSE historico), gastos legislativos detalhados, CPIs, DOU completo, Querido Diario. Objetivo: rede Politico -> Doador -> Empresa -> Contrato -> Sancao.

Fase 3 — Corpo Empresarial (em andamento)

CNPJ completo (53.6M empresas em carregamento), ComprasNet/PNCP, BNDES, RAIS, CVM, BCB. Objetivo: mapear toda a economia e cruzar com politica.

Fase 4 — Analise Avancada (proximo)

Entity Resolution (fuzzy matching), PageRank, Community Detection (Louvain), Anomaly Detection, Timeline Analysis, Geographic Clustering, Red Flag Scoring (60 indicadores GRAS/World Bank).

Fase 5 — Ecossistema Global (futuro)

Interoperabilidade com FollowTheMoney, Aleph (OCCRP), ICIJ DataShare, Serenata de Amor. Exportar entidades no formato FtM para compatibilidade internacional.

Fase 6 — MCP Tools (futuro)

Publicar as 26+ ferramentas OSINT como servidor MCP, permitindo que qualquer IDE ou agente de IA acesse os dados do grafo diretamente.

8. Limitacoes Atuais

Transparencia exige honestidade sobre o que nao conseguimos fazer (ainda):

Cobertura Parcial: Das 108 fontes mapeadas, apenas 36 estao carregadas. Fontes criticas como DataJud (80M processos), PNCP completo, e TCEs estaduais ainda nao estao integradas.
CNPJ em Carregamento: O dataset completo de 53.6M empresas da Receita Federal esta sendo carregado (ETL em andamento). Ate a conclusao, consultas de empresas podem retornar resultados incompletos.
Entity Resolution Basica: Ainda nao temos fuzzy matching avancado. "JOSE DA SILVA" e "JOSE DA SILVA LTDA" podem nao ser reconhecidos como a mesma entidade. Fase 4 abordara isso.
Sem Dados Privados: Nao temos acesso a dados bancarios, fiscais individuais (IR), comunicacoes privadas ou qualquer dado protegido por sigilo. Operamos apenas com dados publicos.
IA Pode Errar: O agente de IA pode interpretar dados incorretamente ou fazer conexoes que nao existem. Sempre verifique as fontes citadas. Padroes detectados sao indicios, nao provas.
Latencia em APIs Externas: Algumas APIs (PNCP, Portal da Transparencia, DataJud) podem estar lentas ou indisponiveis. O sistema tem fallbacks, mas nao garante 100% de disponibilidade.
Servidor Unico: Toda a infraestrutura roda em um unico VPS ($36/mes). Picos de uso podem causar lentidao. Nao temos redundancia geografica.
Equipe de Uma Pessoa: O projeto e mantido por um desenvolvedor solo com auxilio de agentes de IA. Contribuicoes da comunidade sao essenciais para escalar.

10. Como Participar

A EGOS Inteligencia e um projeto open source que precisa da comunidade para crescer:

Desenvolvedores

Contribua com codigo: pipelines ETL para novas fontes, melhorias no frontend, algoritmos de deteccao. Veja issues abertas no GitHub com tags "good first issue" e "help wanted".

Jornalistas Investigativos

Use a plataforma para investigacoes. Reporte bugs, sugira fontes, compartilhe metodologias. O chat IA pode cruzar dados em segundos que levariam dias manualmente.

Pesquisadores e Academicos

Proponha novos algoritmos de deteccao, valide padroes encontrados, publique papers usando os dados. A API e aberta para pesquisa.

Cidadaos

Use, teste, reporte problemas. Compartilhe com amigos. Quanto mais pessoas usarem, mais rapido encontramos problemas e melhoramos.
